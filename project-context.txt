PROJECT OVERVIEW
================
Material Library KC is a comprehensive construction materials platform for India,
built as a React 18 + TypeScript + Vite + Tailwind CSS web application. It serves
as a LinkedIn + UrbanCompany + Behance hybrid for the construction industry —
connecting brands, studios/consultants, and individual professionals/students.

TECH STACK
==========
- React 18.3.1 + TypeScript
- React Router 7.13.0 (client-side routing via createBrowserRouter)
- Vite 6.3.5 (build tool)
- Tailwind CSS 4.1.12 (@tailwindcss/vite)
- Radix UI (accessible component primitives — 25+ packages)
- lucide-react 0.487.0 (icons)
- MUI Material 7.3.5 (supplementary components)
- Recharts 2.15.2 (charts)
- motion 12.23.24 (animations)
- react-hook-form 7.55.0 (forms)
- embla-carousel-react (carousels)
- CVA + clsx + tailwind-merge (styling utilities)
- date-fns, sonner (utilities)
- No backend — all mock data in TypeScript files

================================================================================
KEY DESIGN DECISIONS & PATTERNS
================================================================================

1. Component Pattern: Functional components with hooks, no class components
2. Styling: Inline styles with CSS vars + Tailwind utility classes + glass.css utilities
3. Routing: React Router 7 with nested layouts using Outlet pattern
4. Dashboard layouts: Collapsible sidebar (desktop) + mobile overlay drawer + top bar
5. Data: All mock data in TypeScript files, no API calls
6. Icons: lucide-react primary, @mui/icons-material secondary
7. No authentication: All dashboards accessible, profile dropdown shows all 3 roles
8. Export convention: Named exports preferred, some legacy default exports
9. Pages wrap in min-h-screen div with Navbar + Footer (public) or Outlet (dashboard)
10. Interactive state: useState for all form/toggle/modal state
11. Glass UI: backdrop-filter blur + semi-transparent backgrounds + inner glow shadows
12. Professionals page: Warm neutral bg (#f8f8f6), dark accents (#1a1a1a), Behance-style masonry
13. Services page: Split layout (category sidebar + feed), discovery-first, glass cards
14. Resources/Home: Category cards with glass frost effect on hover
